# ros2-distributed-communication-demo
ROS2-based multi-agent communication framework using UDP transport.

This project demonstrates a lightweight communication layer
for distributed robotic systems using ROS2 and UDP transport.

It focuses on:
- Peer discovery
- Reliable message exchange
- Latency monitoring
- Modular ROS2 node design

The goal is to simulate coordination infrastructure required
in multi-agent robotic environments.
